## New Feature Request - Workspace and Projects

Workspace -

Workspace are the combination of project, means inside a workspace there can be multiple projects. 

Featuers - 
- Admin can create workspace (name, description, and other fields)
- From the top side bar user can switch workspace and the whole thing will be changes. 
- Once user login into system, the first screen they will see to select a workspace (for Manager and Developer role) and if the user is admin then there will be on more option to add workspace. 


Projects - 

Projects are inside the workspace. We can create projects inside a workspace. Every Project will have different tasks and all (that will do later on), goals and everything.

Features: - 
- Project can be added inside workspace.
- Project owner (who created project) and admin can add people to project.
- Once people will be added to project means once they will login they will access the projects through workspace.
