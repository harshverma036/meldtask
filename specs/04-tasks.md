### Feature Request - Task Management

Inside every Project there will be multiple tasks which we can add and manage them accordinlgy. 

Features list: - 

- any user role user will be able to add and manage task. 
- inside every main task user can add multiple sub task, same for inside any sub-task we can add task. so chain can be any long. 
- inside every task or sub-task there can me different field and all 
    - title
    - description
    - assigned_to
    - due date
    - timestamps (createdAt and updatedAt)
    - description
    - comments (can mention anyone who has access to this project, upload files (any))
    - priority
    - assigned by
    - status (while creating project there user will add multiple card statuses, so that should be visible here only)
- every task will contains some attached assets in which user can uplod file (any + multiple) and attach links. 
- when click on every task the whole model big model should be opened. at the top right corner there will be two options (1. cross sigh and full screen when user click on that the page will open to that task only.)
- at the top of any task model or task page, the breadcrumb menu should be there.
